position

정의 : 박스를 원하는 위치에 배치하는 방법

종류 : static, relative, absolute, fixed

position : static
원래 그대로의 상태, 움직이거나 겹칠 수 없다.
top, right, bottom, left 사용안됨.
position:relative
원하는 위치에 배치, 박스 자리가 유지됨
기준점 : 현재 위치
