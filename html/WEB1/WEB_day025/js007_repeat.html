제어문2(for/while/do while)

        STEP1 for  - 반복횟수 알때
        for(  초기문; 조건문; 증감문 ){  }   
         
        STEP2 while- 반복횟수 모를때
        초기문; 
        while(  조건문 ){     증감문;  }   

        STEP3 do while- 한번은 실행을 해야할때
        초기문; 
        do{     증감문;  } while(  조건문 );  
        
        STEP4 향상된 for
        for( index  in 객체 ){   }
