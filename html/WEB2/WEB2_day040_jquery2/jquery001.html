JQUERY1 - SELCTOR

        1. 전체 선택자 : $("")
        2. 태그 선택자 : h3 선택 $("h3")
        3. 아이디 선택자 : 아이디가 son인 것 선택 $("#son")
        4. 클래스 선택자 : $(".myselector")  클래스가 myselector 인 것 선택
        5. 자식들 선택자 : $("ul li") ul 태그 안에 li들 // $("ul>li") ul 태그 직계자식 li들 
        6. 부모 선택자 : $("son").parent() #son 의 부모
        7. 이전 선택자 : $("son").prev() #son 의 이전
        8. 다음 선택자 : $("son").next() #son 의 다음
        9. 형제 선택자 : $("son").siblings("li") #son 의 형제들
        10. 이벤트 선택자 : $(this)
