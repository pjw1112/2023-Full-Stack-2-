이벤트연결 종류
  1. bind - unbind : 정적연결
  2. on - off : 정적연결 + 동적연결 ★
  3. one : 한번만
  4. delegate : 동적연결만
1) bind
이벤트를 여러개를 연결할때 / jquery 1.9버젼 미만
2) on : 권장사항
이벤트를 여러개를 연결할때 / jquery 1.7버젼 이상에서 권장사항
3) one
참고 : event.handleObj.handler => one이 한번만 이벤트를 연결함. event.handleObj.handler 를 사용하면 이벤트를 다시 실행함. 이벤트가 연속적으로 실행되지않기 위해서 사용함. 한번이 끝난다음에 다시 등록을 시킴으로써 이벤트가 중첩되지 않음.
